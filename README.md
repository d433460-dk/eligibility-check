# 🏥 Automated Patient Eligibility Verification System

## 📌 Problem Statement
Healthcare organizations often verify patient insurance eligibility manually before appointments. This process is time-consuming and prone to human errors, which can lead to claim denials and administrative burden.

Our system automates patient eligibility verification to ensure insurance coverage is validated before appointments.

## 🎯 Objective
The goal of this project is to build an automated system that:

- Retrieves patient insurance information
- Verifies insurance eligibility automatically
- Flags invalid or expired insurance
- Notifies hospital staff about issues
- Maintains verification logs

## ⚙️ Features
- Patient Registration
- Insurance Eligibility Verification
- Real-time Status Display
- Verification Logs
- User-friendly interface
